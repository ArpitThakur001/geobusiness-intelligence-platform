# 📍 Enterprise-Grade GIS Dashboard for Business Expansion Intelligence

## 📖 Overview

**GeoBusiness Intelligence Platform** is an AI-powered Geographic Intelligence Dashboard designed to help businesses identify high-potential cafe locations using real-world geospatial data, opportunity scoring, clustering, radius analysis, and interactive GIS visualization.

Instead of displaying static business information, the platform transforms raw geographical coordinates into actionable business intelligence.

The application combines modern web technologies with geographic visualization to deliver an enterprise-style analytics platform suitable for startups, franchises, investors, and market research teams.

# 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/ArpitThakur001/geobusiness-intelligence-platform.git
```

### Navigate to Project

```bash
cd geobusiness-intelligence-platform
```

### Install Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
